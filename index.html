HTML stands for HyperText Markup Language.

It is the standard language for creating webpages.

HTML consists of a series of elements.

Elements tell the browser how to display the content.

HTML elements are represented by tags.

Tags label pieces of content such as "heading", "paragraph", "table", etc.

Browsers do not display the HTML tags but use them to render the content.

This is an example of a paragraph element.
